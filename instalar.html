📌 Instalar — arraste pra barra de favoritos

Arraste o botão abaixo para a barra de favoritos:

📊 Monitor Last Mile
⇧ Arraste com o mouse pra cima, na barra de favoritos

🚀 Como usar

  • 1. Arrasta o botão acima pra barra de favoritos
  • 2. Abre o Last Mile do ML (ou qualquer outra página)
  • 3. Clica no favorito → painel aparece flutuando
  • 4. Clicar de novo recarrega o painel com a versão mais nova

💡 Não vê a barra de favoritos?

  • Chrome / Edge / Firefox: Ctrl + Shift + B

🔄 Atualizações automáticas

  • Quando o código for atualizado no servidor, todos os usuários recebem a nova versão na próxima vez que clicarem no favorito.
  • Não precisa reinstalar nada.
